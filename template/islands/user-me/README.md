# user-me

Your personal memory — preferences, style, decisions, working agreements with your agents.
Rename this folder to `user-<your-handle>`.

**Who writes here:** you, and agents recording preferences you state explicitly.

**Conventions:**
- Tag prefix: `user/` (e.g. `user/preference`, `user/decision`).
- Pin everything you state explicitly — these are authoritative for agents.
- Confidence `1.0` for direct statements.

**Common notes:** code style preferences, communication style, agent behavior rules
("never commit without asking", "no co-author trailers", ...).
