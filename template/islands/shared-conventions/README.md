# shared-conventions

Cross-island pinned facts — conventions that apply to *all* islands: git style,
infra endpoints, agent handoff rules.

**Who writes here:** any agent or you.

**Conventions:**
- Tag prefix: `shared/` (e.g. `shared/git`, `shared/infra`).
- Pin everything here — these are canonical, fleet-wide rules.
- Confidence `1.0` for agreed conventions.
- Island-specific notes reference these via `[[wikilinks]]` instead of restating them.
